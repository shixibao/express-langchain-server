# express-langchain-server

## proxy

```bash
export http_proxy=http://127.0.0.1:7890
export https_proxy=http://127.0.0.1:7890
```

## Setup

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```
